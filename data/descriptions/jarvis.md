🛡️ JARVIS v7.6 | Windows API 远程指挥官

项目定义：一个以 API 为核心的 Windows 远程控制引擎，旨在将你的 PC 变为可通过云端或语音操纵的“智能中枢”。

核心用途

iOS/Siri 联动：通过快捷指令实现“嘿 Siri，锁定电脑”等零接触语音控制。

远程自动化：利用标准 HTTP POST 请求，从任何设备远程触发系统级操作。

静默管理：支持开机自启与后台运行，通过系统托盘实现无感交互。

核心功能

指令执行：一键实现锁屏、静音、运行脚本或自定义 Python 动作。

真人播报：集成微软 Edge-TTS，提供电影级的高质量语音回馈。

网页仪表盘：可视化监控状态、测试功能及热重载配置文件。

异步响应：毫秒级指令响应，操作与语音播报互不阻塞。
